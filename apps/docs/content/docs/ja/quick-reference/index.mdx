Simワークフローエディタの日常的な操作のクイックルックアップです。キーボードショートカットについては、[キーボードショートカット](/keyboard-shortcuts)を参照してください。

<Callout type="info">
  **Mod**は、macOSでは`Cmd`、Windows/Linuxでは`Ctrl`を指します。
</Callout>
