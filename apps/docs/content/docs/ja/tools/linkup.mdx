[Linkup](https://linkup.so)は、Simとシームレスに統合される強力なウェブ検索ツールで、AIエージェントが適切な情報源の引用とともに最新のウェブ情報にアクセスできるようにします。

Linkupは、AIエージェントにウェブから最新情報を検索する能力を提供することで機能を強化します。エージェントのツールキットに統合すると：

- **リアルタイム情報アクセス**：エージェントはウェブから最新情報を取得し、応答を最新かつ関連性の高いものに保ちます。
- **情報源の引用**：すべての情報は適切な引用とともに提供され、透明性と信頼性を確保します。
- **簡単な実装**：最小限の設定でエージェントのツールセットにLinkupを追加できます。
- **文脈認識**：エージェントは個性や会話スタイルを維持しながらウェブ情報を活用できます。

Linkupをエージェントに実装するには、エージェントの設定にツールを追加するだけです。これにより、エージェントは最新情報を必要とする質問に答える際にウェブを検索できるようになります。
{/* MANUAL-CONTENT-END */}

## 使用方法

Linkupをワークフローに統合します。ウェブ検索が可能です。APIキーが必要です。

## ツール

### `linkup_search`

Linkupを使用してウェブ上の情報を検索する

#### 入力

| パラメータ | 型 | 必須 | 説明 |
| --------- | ---- | -------- | ----------- |
| `q` | string | はい | 検索クエリ |
| `depth` | string | はい | 検索深度（「standard」または「deep」のいずれかである必要があります） |
| `outputType` | string | はい | 返す出力のタイプ（「sourcedAnswer」または「searchResults」である必要があります） |
| `apiKey` | string | はい | Linkup APIキーを入力してください |
| `includeImages` | boolean | いいえ | 検索結果に画像を含めるかどうか |
| `fromDate` | string | いいえ | 結果をフィルタリングする開始日（YYYY-MM-DD形式） |
| `toDate` | string | いいえ | 結果をフィルタリングする終了日（YYYY-MM-DD形式） |
| `excludeDomains` | string | いいえ | 検索結果から除外するドメイン名のカンマ区切りリスト |
| `includeDomains` | string | いいえ | 検索結果を制限するドメイン名のカンマ区切りリスト |
| `includeInlineCitations` | boolean | いいえ | 回答にインライン引用を追加する（outputTypeが「sourcedAnswer」の場合のみ適用） |
| `includeSources` | boolean | いいえ | レスポンスにソースを含める |

#### 出力

| パラメータ | 型 | 説明 |
| --------- | ---- | ----------- |
| `answer` | string | 検索クエリに対する情報源付きの回答 |
| `sources` | array | 回答の作成に使用された情報源の配列。各情報源には名前、URL、スニペットが含まれます |
