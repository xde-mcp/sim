[BrowserUse](https://browser-use.com/) est une puissante plateforme d'automatisation de navigateur qui vous permet de créer et d'exécuter des tâches de navigateur de manière programmatique. Elle offre un moyen d'automatiser les interactions web grâce à des instructions en langage naturel, vous permettant de naviguer sur des sites web, remplir des formulaires, extraire des données et effectuer des séquences complexes d'actions sans écrire de code.

Avec BrowserUse, vous pouvez :

- **Automatiser les interactions web** : naviguer sur des sites web, cliquer sur des boutons, remplir des formulaires et effectuer d'autres actions de navigateur
- **Extraire des données** : récupérer du contenu à partir de sites web, y compris du texte, des images et des données structurées
- **Exécuter des flux de travail complexes** : enchaîner plusieurs actions pour accomplir des tâches web sophistiquées
- **Surveiller l'exécution des tâches** : observer l'exécution des tâches de navigateur en temps réel avec un retour visuel
- **Traiter les résultats de manière programmatique** : recevoir des sorties structurées des tâches d'automatisation web

Dans Sim, l'intégration de BrowserUse permet à vos agents d'interagir avec le web comme s'ils étaient des utilisateurs humains. Cela permet des scénarios comme la recherche, la collecte de données, la soumission de formulaires et les tests web - le tout grâce à de simples instructions en langage naturel. Vos agents peuvent recueillir des informations à partir de sites web, interagir avec des applications web et effectuer des actions qui nécessiteraient normalement une navigation manuelle, élargissant ainsi leurs capacités pour inclure l'ensemble du web comme ressource.

## Instructions d'utilisation

Intégrez l'utilisation du navigateur dans le flux de travail. Peut naviguer sur le web et effectuer des actions comme si un utilisateur réel interagissait avec le navigateur. Nécessite une clé API.

## Outils

### `browser_use_run_task`

Exécute une tâche d'automatisation de navigateur en utilisant BrowserUse

#### Entrée

| Paramètre | Type | Obligatoire | Description |
| --------- | ---- | ----------- | ----------- |
| `task` | string | Oui | Ce que l'agent de navigateur doit faire |
| `variables` | json | Non | Variables optionnelles à utiliser comme secrets \(format : \{key: value\}\) |
| `format` | string | Non | Pas de description |
| `save_browser_data` | boolean | Non | S'il faut sauvegarder les données du navigateur |
| `model` | string | Non | Modèle LLM à utiliser \(par défaut : gpt-4o\) |
| `apiKey` | string | Oui | Clé API pour l'API BrowserUse |

#### Sortie

| Paramètre | Type | Description |
| --------- | ---- | ----------- |
| `id` | string | Identifiant d'exécution de la tâche |
| `success` | boolean | Statut d'achèvement de la tâche |
| `output` | json | Données de sortie de la tâche |
| `steps` | json | Étapes d'exécution effectuées |

## Notes

- Catégorie : `tools`
